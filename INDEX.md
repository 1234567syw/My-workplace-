## 🚀 Quick Command Reference

```bash
# === FIRST TIME SETUP ===
pip install opencv-python              # Install OpenCV
python test_video_tool.py              # Run setup

# === TESTING ===
python test_video_tool.py              # Validate all videos

# === PROCESSING ===
python example_video_processing.py     # Process with nameplates

# === GITHUB ===
git status                              # Check changes
git add .                               # Add all files
git commit -m "Your message"            # Commit
git push origin main                    # Push to GitHub

# === FILE OPERATIONS ===
cp your_video.mp4 test_videos/          # Add test video
cp video.mp4 camera_feeds/camera_1.mp4  # Add camera feed
ls test_videos/                         # List files
cat config/video_config.json            # View config
```
